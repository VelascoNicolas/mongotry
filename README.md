# API Full Salud 
### Lista de comandos 
1- Clonar el repositorio  
2- Instalar dependencias   
```bash
npm i
```
3- Configurar el `.env` a base de `.env.template`  
4- Correr proyecto 
```bash
npm run start:dev
```

## Refactorizar código
Antes de crear un pull request ejecutar:
- Para verificar errores de linting
```bash
npm run lint
```
- Para formatear el código
```bash
npm run format
```
